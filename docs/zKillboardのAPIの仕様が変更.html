zKillboardのAPIの仕様が変更

Administrator | 2018年10月09日

zKillboardのAPIの仕様が変更されたらしいです。 コミットログとしてはおそらくここです。
"implemented zkb only data for api, and upped limit to 1000 per call"

ドキュメントの方にも既に追記されてます。

/zkbOnly/ (This is now applied to all requests as for 2018-10-02)
「2018年10月2日からはzKillboardのAPIは自分達の方で算出したデータしか出力しません」

これからはどうするの？

どうやら結構前からキルメールの詳細はESIからでも取れてたらしいので、これを使います。

まずは、いつものようにzKillboardの方から取得したいキルメール一覧のJSONを取得します。

以前はここにもキルメールの詳細もあったんですが、今はありません（zKillboardが独自に算出しているPointsやAwox, Soloの判定はzkbの項目にあります）

最後にkillmail_idとhashの値を使って、ESIからキルメールの詳細を取得する感じっぽいです。
